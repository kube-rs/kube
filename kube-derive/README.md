# kube-derive
Add `#[derive(CustomResource)]` to your custom resource struct.

## Installation
Add the `derive` feature to `kube`:

```toml
[dependencies]
kube = { version = "0.75.0", feature = ["derive"] }
```

## Usage
See the **[kube-derive API Docs](https://docs.rs/kube-derive/)**
